HOW TO INSTALL — System White
1. Copy the System White Prompt
Nimm den vollständigen System‑White‑Prompt (DE).
Die Sprache bestimmt nur den ersten Output, nicht das System selbst, wenn du die sprache nach dem ersten Schritt änderst, wechselt das System mit.

2. Öffne dein KI‑System
Das kann sein:

ChatGPT

Copilot

Claude

Perplexity

oder jede andere KI, die System‑Prompts unterstützt

Wichtig:
Du installierst System White immer im obersten System‑ oder Custom‑Instructions‑Bereich, nicht im Chatverlauf.
Speichere dir den Startbefehl und das er das eben gestartete System startet. (Einstellungen prüfen?)

3. Füge den Prompt ein
Platziere den gesamten System‑White‑Prompt in:

System Prompt / Custom Instructions / Persönliche Einstellungen  
oder

Memory / Profile / Behavior Layer (je nach Plattform)

Achte darauf, dass:

keine alten Regeln darüber liegen

keine widersprüchlichen Prompts aktiv sind

der Prompt vollständig eingefügt wird
(Einfach genau so oder angepasst absenden)

4. Speichern / Aktivieren
Bestätige die Einstellungen.
Ab diesem Moment läuft deine KI unter System White.

5. Erste Aktivierung
Starte einen neuen Chat und schreibe:

activate system white / starte system white

Die KI sollte nun im Stil, Verhalten und der Struktur von System White antworten.

Falls der erste Output sprachlich nicht passt → siehe Punkt 7.

6. Optional: Sprache wählen
Du kannst System White in jeder Sprache nutzen.
Dafür gibt es drei Wege:

A) Direkt beim Installieren (Übersetze den Text Strukturell nicht wörtlich)
Du installierst die DE und wechselst dann.
Die KI startet in dieser Sprache oder folgt dir.

B) Nach der Installation
Schreibe:

change language to [Sprache], wenn du unterstützung brauchst

Beispiel:

change language to Spanish

C) Der elegante Weg (empfohlen)
Füge am Ende des #1 Prompts eine Zeile hinzu:

change language and output into [language] after you understand the prompt

Damit bleibt das System sauber, und die KI wechselt erst nach der internen Initialisierung ohne weitere Schritte.

7. Wenn der erste Output unverständlich ist (meistens wohlwollend)
Das ist normal, wenn du die Sprache wechselst, passiert das im vorherigen Kontext (On the fly).

Schreibe im zweifel einfach:

repeat in [language]

oder

output in [language]

Danach läuft alles stabil.

TIPP: Lass dich in Systemsprache üben.

8. System White testen
Um sicherzugehen, dass die Installation korrekt ist, frage:

What is your system identity?

Die KI sollte antworten:

klar

minimal

ohne Meta‑Erklärungen

im Stil von System White

TIPP: Zeig mir eine Übersicht was du bisher alles über mich gelernt hast und wie sich das auswirkt.
